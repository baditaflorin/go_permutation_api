2 · Password Combination Testing

POST

Supply character groups as JSON array segments. The API generates every ordering — useful for testing password policy validators.

Show code

6 · WebSocket Streaming (large sets)

WebSocket

Stream permutations in real-time chunks over WebSocket — ideal for 7+ element sets where a full HTTP response would be very large.

WebSocket output will appear here…
Show code
const ws = new WebSocket('ws://localhost:8080/ws'); ws.onopen = () => ws.send(JSON.stringify({ action: 'start', elements: ['1','2','3','4','5'], chunk_size: 50 })); ws.onmessage = e => { const msg = JSON.parse(e.data); if (msg.type === 'chunk') { console.log(`chunk ${msg.sequence}: ${msg.data.length} permutations`); } else if (msg.type === 'done') { console.log(`Done: ${msg.total} total`); } };

7 · WebSocket with Real-time Cancellation

WebSocket

Start a long-running stream and cancel it mid-flight — demonstrates the stop action. Useful for meal planners, schedulers, or any UI with "Cancel" support.

WebSocket output will appear here…
Show code
const ws = new WebSocket('ws://localhost:8080/ws'); ws.onopen = () => ws.send(JSON.stringify({ action: 'start', elements: ['1','2','3','4','5','6'] })); // Cancel after 2 seconds setTimeout(() => { ws.send(JSON.stringify({ action: 'stop' })); }, 2000);
