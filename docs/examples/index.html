3 · Route Optimization (TSP Brute-force)

WASM GET

Every visit order for a set of cities — feed each into a distance function to find the shortest tour.

Results will appear here…
Show code
const routes = goPermutations(["NYC","LA","CHI","MIA"]); let best = Infinity; routes.forEach(r => { const d = totalDistance(r); if (d < best) best = d; });

6 · WebSocket Streaming (requires server)

WebSocket

Stream permutations in real-time chunks — best for 7+ element sets. Requires a running API server (WebSocket cannot run in WASM mode).

WebSocket output will appear here…
Show code
const ws = new WebSocket('ws://localhost:8080/ws'); ws.onopen = () => ws.send(JSON.stringify({ action: 'start', elements: ['1','2','3','4','5'], chunk_size: 50 })); ws.onmessage = e => { const msg = JSON.parse(e.data); if (msg.type === 'done') console.log(`Done: ${msg.total} total`); };

7 · WebSocket with Real-time Cancellation

WebSocket

Start a stream and cancel it mid-flight via the stop action. Requires a running API server.

WebSocket output will appear here…
Show code
const ws = new WebSocket('ws://localhost:8080/ws'); ws.onopen = () => ws.send(JSON.stringify({ action: 'start', elements: ['1','2','3','4','5','6'] })); setTimeout(() => ws.send(JSON.stringify({ action: 'stop' })), 2000);
