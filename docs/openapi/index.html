This is the interactive OpenAPI 3.0 specification for the Go Permutation API. To try the "Try it out" button, run a local server first: docker run -p 8080:8080 -e SERVER_HOST=0.0.0.0 ghcr.io/baditaflorin/go_permutation_api --serve then set the server URL to http://localhost:8080.
